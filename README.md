基于antd的中台应用组件扩展库

## Getting Started

Install dependencies,

```bash
yarn add @rezerojs/components  # or npm i @rezerojs/components -S
```

Usage

```bash
import { IndicatorCard } from '@rezerojs/components';
```

<br />

更多使用方式, 请参见[Components](/components)下具体组件示例及 API 文档
